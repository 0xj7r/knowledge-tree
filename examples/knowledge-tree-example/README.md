## How to read this tree

1. Start here. Follow into the subdir that matches your question.
2. Every subdir has its own `README.md` index table. Use it as a map.
3. Individual nodes link out via the `related:` frontmatter field. Follow those links when they appear relevant.
4. If the answer is not in the tree, it is not yet known. Do not fabricate.
